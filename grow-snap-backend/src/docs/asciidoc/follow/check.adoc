==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/follows/check/{followingId} \
  -H "Authorization: Bearer {access_token}"
----

==== Response Values

* `isFollowing: true` - 팔로우 중
* `isFollowing: false` - 팔로우하지 않음
