==== Notes

* 자기 자신을 팔로우할 수 없습니다
* 이미 팔로우 중인 사용자는 다시 팔로우할 수 없습니다
* 팔로우 성공 시 양쪽 사용자의 팔로워/팔로잉 수가 자동으로 증가합니다
