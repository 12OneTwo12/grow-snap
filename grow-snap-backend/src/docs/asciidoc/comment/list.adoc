==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/videos/{videoId}/comments \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 댓글은 계층 구조로 반환됩니다 (부모 댓글 -> 대댓글)
* 삭제된 댓글은 목록에 포함되지 않습니다
* 댓글이 없는 경우 빈 배열을 반환합니다
* 인증된 사용자만 조회할 수 있습니다
