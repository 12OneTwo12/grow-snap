==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/videos/{videoId}/comments \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "좋은 영상입니다!",
    "timestampSeconds": 120,
    "parentCommentId": null
  }'
----

==== Notes

* 댓글 내용은 필수이며 비어있을 수 없습니다
* timestampSeconds는 선택 사항으로, 특정 시점에 대한 댓글인 경우 사용합니다
* parentCommentId를 지정하면 대댓글(답글)로 작성됩니다
* 인증된 사용자만 댓글을 작성할 수 있습니다
