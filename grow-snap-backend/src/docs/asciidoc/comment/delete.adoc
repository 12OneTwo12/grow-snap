==== Example

[source,bash]
----
curl -X DELETE http://localhost:8080/api/v1/comments/{commentId} \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 자신이 작성한 댓글만 삭제할 수 있습니다
* 삭제된 댓글은 복구할 수 없습니다 (Soft Delete 적용)
* 존재하지 않는 댓글 삭제 시 404 Not Found를 반환합니다
* 다른 사용자의 댓글 삭제 시 403 Forbidden을 반환합니다
