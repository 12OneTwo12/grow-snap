==== Example

[source,bash]
----
curl -X PATCH http://localhost:8080/api/v1/profiles \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "nickname": "updatednick",
    "bio": "수정된 자기소개"
  }'
----

==== Notes

* 모든 필드는 선택사항입니다
* 제공된 필드만 업데이트됩니다
* `nickname` 변경 시 중복 검사가 수행됩니다
