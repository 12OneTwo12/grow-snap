=== 저장한 콘텐츠 목록 조회

사용자가 저장한 콘텐츠 목록을 조회합니다.

==== Request

include::{snippets}/save-list/http-request.adoc[]

==== Response

include::{snippets}/save-list/http-response.adoc[]
include::{snippets}/save-list/response-fields.adoc[]

==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/users/me/saved-videos \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 자신이 저장한 콘텐츠만 조회할 수 있습니다
* 저장한 콘텐츠가 없는 경우 빈 배열을 반환합니다
* 저장된 순서대로 반환됩니다 (최근 저장한 순)
* 삭제된 콘텐츠는 목록에 포함되지 않습니다
