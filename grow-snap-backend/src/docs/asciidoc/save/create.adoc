==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/videos/{videoId}/save \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 이미 저장한 콘텐츠는 다시 저장할 수 없습니다
* 저장 성공 시 콘텐츠의 저장 수가 자동으로 증가합니다
* 저장한 콘텐츠는 나중에 보기 위해 북마크한 것으로 간주됩니다
* 인증된 사용자만 콘텐츠를 저장할 수 있습니다
