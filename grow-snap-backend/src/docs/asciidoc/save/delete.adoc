==== Example

[source,bash]
----
curl -X DELETE http://localhost:8080/api/v1/videos/{videoId}/save \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 저장하지 않은 콘텐츠는 취소할 수 없습니다
* 취소 성공 시 콘텐츠의 저장 수가 자동으로 감소합니다
* 인증된 사용자만 저장을 취소할 수 있습니다
